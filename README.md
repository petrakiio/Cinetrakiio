# CineTrakio

Site estático em HTML, CSS e JavaScript.

## Estrutura

- `index.html`: entrada rápida do site.
- `pages/home/`: páginas principais, como início, login, cadastro, novidades e sobre.
- `pages/generos/`: páginas de listagem por gênero.
- `pages/filmes/`: páginas individuais dos filmes, separadas por categoria.
- `assets/css/`: folhas de estilo.
- `assets/js/`: scripts do site.
- `assets/img/`: imagens e logos.
- `docs/`: materiais e anotações do projeto.

Para abrir o site, use `index.html` na raiz do projeto.
